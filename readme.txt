== Description ==

Connect user account with LINE ID

== Installation ==

You can put lineconnect folder to plugins folder and activate plugin. save channel access token and channel secret on the setting page.
Webhook URL setting on LINE Developer should be 'https://your-domain/wp-content/plugins/lineconnect/bot.php
